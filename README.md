# Pequeno site — Revolução Industrial

Este projeto é um exemplo simples de uma página sobre a Revolução Industrial contendo três imagens vetoriais (SVG) e uma interação de modal para ver cada imagem em tamanho maior.

Arquivos principais:
- `index.html` — página principal com conteúdo e modal
- `styles.css` — estilos da página
- `main.js` — comportamentos (abrir/fechar modal)
- `assets/` — três imagens SVG (spinning, factory, steam)

Como abrir localmente:
1. Abra o `index.html` em um navegador (duplo clique ou arraste para o navegador).
2. Clique em qualquer imagem para ver em tamanho maior. Use `Esc` para fechar ou clique fora da imagem.

Melhorias possíveis:
- Substituir os SVGs por fotos históricas (lembre-se dos direitos autorais) ou imagens públicas (ex: Unsplash).
- Adicionar navegação entre imagens no modal (próxima/anterior).
- Tornar a página multilíngue.
